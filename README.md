AI Chat Session App:

A full-stack AI chat application built using React (Vite) and Node.js + Express,
featuring persistent chat sessions, mock AI responses, and structured table outputs.
Backend is deployed on Render and frontend on Vercel.


Live Links:
GitHub Repository: https://github.com/Deepak-Sl/ai-chat-app
Backend (Render): https://ai-chat-backend-tcel.onrender.com
Frontend (Vercel): (Add your deployed link after deployment)


Features:
- Session System: Create new chats, view history, persistent storage
- Chat Functionality: Mock AI responses, timestamped messages, table results
- Frontend: React + Vite, Axios, Router, responsive UI
- Backend: Node.js, Express, CORS, mock AI engine

Tech Stack:
Frontend: React (Vite), Axios, React Router, CSS
Backend: Node.js, Express, CORS
Setup:
Clone repo → run backend → run frontend → update BASE_URL.

API Endpoints:
GET /api/sessions
GET /api/new-chat
GET /api/session/:id
POST /api/chat/:id
